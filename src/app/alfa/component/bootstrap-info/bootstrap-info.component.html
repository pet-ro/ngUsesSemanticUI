Hier erhalten Sie Information zum Starten des JSON Servers.

Starten des JSON Servers:

  • Neues Terminal anlegen für die untenstehenden Commandos.
  • JSON Server starten mit dem Befehl 
    
  node_modules/.bin/json-server --watch  src/app/alfa/model/data/alfa.json
  • Gegebenfalls `.json` file validieren. Zum Beispiel via jsonlint.com.
  • Erfolgreicher Json Server listet die zur verfügungstehende API: 
    
      Done

      Resources
      http://localhost:3000/designs
      http://localhost:3000/services
      http://localhost:3000/news
      http://localhost:3000/teams
    
      Home
      http://localhost:3000
